# ShowYourPrefix
一个简简单单基于LP插件头衔功能的玩家自行修改头衔的插件（支持违禁词）
这是我做的第一个插件，闲的没啥事发一下，希望有几个人看见
这个插件是基于LP插件的头衔功能，请确保你的服务器已经安装lp并且配置了头衔功能
理论上1.21.版本都能用吧...目前只测试过1.21.8paper可以使用 其他的不知道
之后可能会更新GUI 让一些想不出头衔名字的玩家选择一个头衔 但那就不知道啥时候了
一些指令
/cplt --申请头衔
管理员一些指令
/cplt on/off 玩家ID -- 启用或关闭玩家头衔申请
/colt list --列出被禁止申请的玩家
/cplt reload -- 你猜
